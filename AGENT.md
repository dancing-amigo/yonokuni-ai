# Yonokuni AI Agent Documentation

プロジェクトの実装ドキュメントは `.agent/docs/` 以下に整理されています。目的に応じて以下のファイルを参照してください。

- `.agent/docs/architecture.md` — 全体アーキテクチャと主要コンポーネント
- `.agent/docs/game_rules.md` — ゲームルールと実装要点
- `.agent/docs/training_pipeline.md` — 自己対戦生成と学習の流れ
- `.agent/docs/evaluation_and_gating.md` — 評価スクリプトとゲーティング
- `.agent/docs/cli_tools.md` — 各種 CLI ツールの使い方
- `.agent/docs/inference_and_export.md` — 推論ユーティリティとエクスポート方法

必要に応じて `.agent/docs/` 内のファイルを拡張・更新し、ナレッジの共有を続けてください。
